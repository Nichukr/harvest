Installation from source
==========

Prerequisites
-------------
1. harvest-tools (see http://harvest.readthedocs.org/en/latest/content/harvest-tools/installation.html)
2. libMuscle
3. Python 2.6+

Download source and compile
---------------
1. git clone github.com/marbl/parsnp parsnp_src
2. cd parsnp_src
3. ./configure
4. make install
